# Точки роста
## 1# Генерация сопроводительного письма
### Из резюме нужно доставать:
1. ФИО
2. Стаж
3. Проектный опыт
4. Технические навыки (Hard skills)
5. Личные качества (Soft skills)

### Из вакансии нужно доставать:
1. Название проекта
2. Задачи
3. Технические навыки (Hard skills)
4. Личные качества (Soft skills)

## 2# Первичная фильтрация резюме
### Из вакансии и резюме фильтруем по:
- формат работы (удаленка, гибрид, офис)
- тип занятости (полная, частичная, проектная, подработка)
- варианты оформленя (Трудовой договор, Договор ГПХ с самозанятым, Договор ГПХ с ИП, Договор ГПХ с физлицом)
- технические навыки (Hard skills)

## 3# Улучшение системы
### Тесты:
#### 1 - Автогенерация 4 тестов для 4 видов резюме:
- Релевантное
- Нерелевантное
- Полупустое
- Пустое

При нажатии кнопки "Запуск Тестов" генерируется 4 сопроводительных письма для такого-то промпта с такой-то моделью.
